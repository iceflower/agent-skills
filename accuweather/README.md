# accuweather

AccuWeather 기반 날씨 정보 조회 규칙 (일기예보, 미세먼지, 자외선 지수 등)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r accuweather/ ~/.agents/skills/accuweather/
```

### 프로젝트 설치

```bash
cp -r accuweather/ .agents/skills/accuweather/
```
