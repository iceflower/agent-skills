# code-quality

코드 품질 6대 원칙 (가독성, 예측성, 오용 방지, 모듈성, 재사용성, 테스트 용이성)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/code-quality
cp SKILL.md ~/.agents/skills/code-quality/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/code-quality
cp SKILL.md .agents/skills/code-quality/
```
