# git-workflow

Git 커밋 컨벤션 및 브랜치 전략

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r git-workflow/ ~/.agents/skills/git-workflow/
```

### 프로젝트 설치

```bash
cp -r git-workflow/ .agents/skills/git-workflow/
```
