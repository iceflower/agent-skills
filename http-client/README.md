# http-client

외부 API 클라이언트 규칙 + Spring Boot 구현 (RestClient, Resilience4j)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r http-client/ ~/.agents/skills/http-client/
```

### 프로젝트 설치

```bash
cp -r http-client/ .agents/skills/http-client/
```
