# java-kotlin-interop

Java-Kotlin 상호 운용 가이드 (JSpecify, JVM 어노테이션, 코루틴 브릿지)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r java-kotlin-interop/ ~/.agents/skills/java-kotlin-interop/
```

### 프로젝트 설치

```bash
cp -r java-kotlin-interop/ .agents/skills/java-kotlin-interop/
```
