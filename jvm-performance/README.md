# jvm-performance

JVM 성능 튜닝 (GC 알고리즘, 힙 분석, 프로파일링)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r jvm-performance/ ~/.agents/skills/jvm-performance/
```

### 프로젝트 설치

```bash
cp -r jvm-performance/ .agents/skills/jvm-performance/
```
