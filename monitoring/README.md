# monitoring

옵저버빌리티 패턴 + Spring Boot 구현 (Actuator, Micrometer, 분산 트레이싱)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r monitoring/ ~/.agents/skills/monitoring/
```

### 프로젝트 설치

```bash
cp -r monitoring/ .agents/skills/monitoring/
```
