이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/mysql
cp SKILL.md ~/.agents/skills/mysql/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/mysql
cp SKILL.md .agents/skills/mysql/
```
