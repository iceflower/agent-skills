# pdf-handling

PDF 파일 읽기 및 처리 규칙 (pdftoppm 변환)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r pdf-handling/ ~/.agents/skills/pdf-handling/
```

### 프로젝트 설치

```bash
cp -r pdf-handling/ .agents/skills/pdf-handling/
```
