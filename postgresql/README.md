# postgresql

PostgreSQL 규칙 (JSONB, GIN 인덱스, CTE, 파티셔닝, 확장)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/postgresql
cp SKILL.md ~/.agents/skills/postgresql/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/postgresql
cp SKILL.md .agents/skills/postgresql/
```
