# resume

개발자 이력서 작성 가이드 (STAR 메서드, 스킬 정리, 프로젝트 기술)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/resume
cp SKILL.md ~/.agents/skills/resume/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/resume
cp SKILL.md .agents/skills/resume/
```

## 호환 도구

| 도구 | 전역 경로 | 프로젝트 경로 |
|------|-----------|---------------|
