이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r security/ ~/.agents/skills/security/
```

### 프로젝트 설치

```bash
cp -r security/ .agents/skills/security/
```
