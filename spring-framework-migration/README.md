# spring-framework-migration

Spring Framework 버전 마이그레이션 (5.x → 6.x → 7.0)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/spring-framework-migration
cp SKILL.md ~/.agents/skills/spring-framework-migration/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/spring-framework-migration
cp SKILL.md .agents/skills/spring-framework-migration/
```
