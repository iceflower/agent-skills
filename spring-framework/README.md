# spring-framework

Spring Framework 핵심 규칙 + Spring Boot 구현 패턴 + Framework/Boot 버전 마이그레이션 가이드

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r spring-framework/ ~/.agents/skills/spring-framework/
```

### 프로젝트 설치

```bash
cp -r spring-framework/ .agents/skills/spring-framework/
```
