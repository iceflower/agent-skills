# spring-jpa

Spring Data JPA (N+1 방지, @Transactional, 엔티티 컨벤션)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r spring-jpa/ ~/.agents/skills/spring-jpa/
```

### 프로젝트 설치

```bash
cp -r spring-jpa/ .agents/skills/spring-jpa/
```
