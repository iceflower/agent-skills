# spring-webflux

Spring WebFlux & Kotlin Coroutines (R2DBC, Flow, WebClient, SSE)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/spring-webflux
cp SKILL.md ~/.agents/skills/spring-webflux/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/spring-webflux
cp SKILL.md .agents/skills/spring-webflux/
```
