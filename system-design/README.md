# system-design

대규모 시스템 설계 패턴 (샤딩, 캐싱, CDN, 메시지 큐)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/system-design
cp SKILL.md ~/.agents/skills/system-design/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/system-design
cp SKILL.md .agents/skills/system-design/
```
