# testing-unit

BDD 스타일 단위 테스트 (Describe-Context-It, 테스트 더블)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
mkdir -p ~/.agents/skills/testing-unit
cp SKILL.md ~/.agents/skills/testing-unit/
```

### 프로젝트 설치

```bash
mkdir -p .agents/skills/testing-unit
cp SKILL.md .agents/skills/testing-unit/
```
