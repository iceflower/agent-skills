# testing-unit

BDD 스타일 단위 테스트 + 통합 테스트 (Testcontainers, Mock 서버)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r testing-unit/ ~/.agents/skills/testing-unit/
```

### 프로젝트 설치

```bash
cp -r testing-unit/ .agents/skills/testing-unit/
```

## 디렉토리 구조

```text

├── SKILL.md
├── README.md
└── references/
    └── integration.md
```
