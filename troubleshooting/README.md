# troubleshooting

트러블슈팅 가이드 (느린 API, 배포 롤백, 연결 문제, 디버깅 원칙)

## 사용 방법

이 스킬은 [Agent Skills 오픈 표준](https://agentskills.io)을 따릅니다.

### 전역 설치

```bash
cp -r troubleshooting/ ~/.agents/skills/troubleshooting/
```

### 프로젝트 설치

```bash
cp -r troubleshooting/ .agents/skills/troubleshooting/
```
